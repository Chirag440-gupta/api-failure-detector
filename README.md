# API Failure Detector

A Spring Boot based monitoring system that continuously checks API health, tracks response status, and detects failures in real time.

## Features
- API Health Monitoring
- Failure Detection
- REST APIs
- MySQL Database
- Spring Boot Backend
- Response Status Tracking

## Tech Stack
- Java
- Spring Boot
- MySQL
- Maven
- REST API

## Future Improvements

- Response Time Analytics
- Scheduled Health Checks
